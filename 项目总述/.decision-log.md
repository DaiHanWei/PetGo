**preview-core-pages.html 组件/交互提取**：见 `mockups/preview-extraction.md`（子任务产出，54 页清单 + 按钮/卡片/徽章/底部抽屉/Tab栏完整规格 + token 例外规则）

**裁定的设计冲突：**

1. **#7D45F6 不采纳**：preview-new-splash-auth-0623.html 探索了一版新主色 #7D45F6（启动页/登录页/登录墙），用户明确裁定不采纳，全部改回全局 token 主色 `#845EC9`。
2. **该文件其余设计模式照单全收**（仅颜色改回 #845EC9）：
   - 狗+猫跑动 logo 动效（dogEnter 入场 + catRun 6 点跑动路径 + 落位变色负形镂空 + wordmark/tagline 渐显 + 圆环 loader），含"当日已播放过则跳过动效直接显示终态"的行为（localStorage 按日期 key）
   - Apple 登录按钮规范：与 Google 同高同圆角同字重、iOS 优先展示在 Google 之上（App Store Review Guideline 4.8），Android 端隐藏整块、只显示 Google（FR-44）
   - 账号注销二步确认模式（PDP 法规驱动）：Step1 后果说明列表 + "继续"，Step2 必须输入"NONAKTIFKAN"原文才能启用提交按钮
3. **底部抽屉动画标准**：三份文件三种写法，裁定采用 cs-contact-sheet.html 的双向写法——开/关都有滑动动画（`transform:translateY(100%↔0)` + `transition .32s cubic-bezier(.32,.72,0,1)`，class 切换 `.open`），而非 preview-core-pages.html 的"只有开、没有关"单向 `animation:slideUp`。
4. **字号刻度**：tokens.json 理想化刻度（28/20/16/14/12）与 preview-core-pages.html 实测刻度（26/19/15/13/11）二选一，用户用对比页（`mockups/typography-comparison.html`）目测后选定 **tokens.json 理想刻度**，新组件一律按 4px 倍数标准刻度走。
5. **圆角刻度**：与字号相反，用户用对比页（`mockups/radius-comparison.html`）目测后选定**保留原型实测值**（按钮14/输入框11/大卡20/抽屉顶角24），不强行收敛到 sm4/md8/lg12/xl16/2xl28/3xl32 的抽象刻度——`rounded` token 改为按真实出现的档位定义（sm6/md12/lg14/xl16/2xl20/3xl24/4xl32/full）。
