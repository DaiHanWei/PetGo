# TailTopia · 6个月产品 Roadmap

**更新日期：** 2026-06-14
**发布节奏：** 每2周一个版本（1.0.0除外）
**支付：** 合作方接入，不占用工程瓶颈
**整体策略：** 月1-2拉新 → 月2开始收费 → 月3-4留存功能全部就位 → 月5运营主导 → 月6收支平衡

## 月1 · 🚀 1.0.0 上线（4-5周）

**目标：** 产品上线，跑通核心两大模块 + 基础社交互动

### 功能清单

- **Auth & Onboarding：** Google登录 · 未登录首页浏览（游客模式，无强制拦截）· 软性登录推荐（浏览至第3页触发）· 昵称确认 · 宠物状态选择（A/B/C）· 宠物档案创建引导（可跳过，前3次重启提示）
- **App 导航：** 5位置底部Tab Bar（首页 / 成长档案 / [+]发布 / 问诊 / 我的）· 未登录点击非首页Tab触发登录引导
- **首页 Feed：** 内容流（时间倒序/无限滚动）· 按宠物状态硬过滤（非算法推荐）· 种子内容冷启动
- **AI 图文分诊：** 照片上传（≤3张/10MB）· 绿/黄/红三级评级 · 条件倒计时协议（黄色）· 半屏强提醒（红色，5秒锁定，不推荐医院）
- **专业问诊：** 兽医聊天（三方SDK）· 兽医账号独立登录入口（账号密码，运营后台创建）· 兽医工作台（待接单/进行中/历史/我的）· 抢单模型（先到先得，后端原子写入）· 1分钟接单超时机制 · 退单机制（最多2次，超出转运营处理）· AI上下文传递给兽医 · 问诊历史记录（AI问诊 + 兽医咨询记录）
- **成长档案：** 宠物档案创建与编辑 · 快乐时刻时间线视图（默认，含第一条记录特殊标记）+ 日历视图（月份格子，有照片显示缩略图）· 里程碑进度条入口 · 档案统计栏（快乐时刻条数/问诊次数）· 问诊记录存档（用户确认）
- **宠物第一次里程碑系统：** 平台固定清单（猫30个 / 狗30个 / 其他宠物15个）· 三级庆祝动效（S半屏 / M全屏+徽章 / L开宝箱式）· 徽章图标打卡交互（关联已发布内容或去发布）
- **宠物名片 H5：** 对外分享页 · Hero区（陪伴天数）· 里程碑徽章条 · 故事数字 · 最近里程碑动态 · 最近5条快乐时刻照片流 · 双CTA（有宠/无宠差异化）· Open Graph标签
- **内容发布：** 统一「+」入口 · 三类内容（日常分享/专业科普/成长日历）· 图片上传（每帖≤9张/单张≤10MB）· 1000字符上限 · 三方自动审核（文字关键词+图片识别，秒级拦截）
- **内容详情页：** 全屏详情 · 迷你主页预览卡（底部弹出，不跳转新页）
- **内容互动：** 点赞（可取消）· 一级评论 + 回复评论（两级）· 内容删除（作者可删，V1.0.0 无编辑功能）· 内容举报（5类选项，进运营审核队列）
- **个人中心（我的）：** 用户信息（头像/昵称可修改）· 宠物卡片 · 我的发布（三类内容时间线倒序）· 宠物状态修改 · 账号注销（印尼PDP合规，二次确认，不可恢复）· 退出登录
- **推送通知：** 兽医回复提醒 · 兽医端新请求提醒 · 被点赞/评论提醒 · 宠物生日提醒（生日前1天）· 陪伴纪念日提醒（满30/100/365天）· L级里程碑解锁提醒 · 权限申请时机：首次问诊后 OR 档案创建后取最早（仅申请一次）
- **系统功能：** App内通知中心（铃铛）· 语言切换（印尼语/英语）· 深链接（7种推送类型全覆盖直跳目标页）· App版本更新提醒（App内弹窗，无需推送权限）
- **兽医评分系统：** 问诊结束后用户评分（1-5星）· 30分钟保护窗口 · 未评分进入问诊Tab时弹出补评一次

> **WhatsApp / SMS OTP登录：** 根据上线后注册转化率数据决定上线时机，不固定在某版本

### 核心指标
- 下载量 · 注册用户数 · AI问诊次数 · 宠物名片分享→下载转化率 · D7留存率

## 月2 · 收费启动

### 1.1.0（月2 第6-7周）
**支付 + 收费正式上线**

- **支付接入：** GoPay / OVO（合作方接入）
- **AI问诊收费：** Rp18,000/次；**每个月第一次AI问诊免费**（降低首次尝试门槛）
- **兽医咨询收费：** Rp30,000/次；AI问诊费可抵扣，升级补差价 Rp12,000
- **Apple登录：** iOS App Store要求
- **健康档案数据录入（早期版）：** 用户记录健康事件类型（疫苗/驱虫/月经/绝育/自定义）+ 上次时间；数据存储供 1.3.0 提醒功能使用；录入成功后提示「开通会员后自动提醒下次时间」，不留歧义
- **成长档案月度回顾：** 每月自动生成月度回顾卡片，汇总当月快乐时刻数、里程碑完成数、最受欢迎照片；用户进入成长档案时展示「[宠物名] 的 X 月」；可一键分享
- **新用户快速解锁里程碑（6个）：** 复用 1.0.0 里程碑庆祝框架（三级动效+徽章），新增6个触发条件；目标是让用户在前两周内积累成就感，提升付费转化率
  - 发布第一篇内容（Day 1-3）
  - 完成第一次AI问诊（Day 1-7）
  - 宠物档案完整度100%（Week 1）
  - 记录第5条快乐时刻（Week 1-2）
  - 获得第一个点赞（Day 1-7）
  - 在平台待满7天（注册后第7天自动触发）
- **首页宠物每日情绪彩蛋便签（FR-11 · The Talking Widget）：** 欢迎语「Hai, Aurel 🐾」下方设置宠物对话气泡便签；每次冷启动或下拉刷新随机抽取一句文案展示，文案库 100% 本地预置，无网络亦正常渲染。
  - **视觉规范（FR-11A）：** 微粉色气泡底板（`color.light.background.active`，`radius.md = 8px`，左上角气泡尾巴）；左侧展示发言者头像：状态 A 已建档案→用户宠物头像，否则→系统 3D 萌化占位头像；正文 `font.family.body / 14px / regular / text.primary`。
  - **随机刷新逻辑（FR-11B）：** 冷启动或下拉刷新时从对应文案子库中 `Random(index)` 抽取；防重复机制：连续两次抽取索引不得相同。
  - **本地文案库（FR-11C）：** 按用户宠物状态硬过滤子库——**状态 A 专用（8 条 Jaksel 混杂风）**：傲娇/撒娇/戏精/雅加达南区名媛口吻（含深夜 23:00-04:00 高概率抽取专属条目 A-08）；**状态 B/C 通用（4 条温暖欢迎风）**：平台虚拟吉祥物口吻，引导看 Feed、渴望云养宠。
  - **边缘情况（FR-11D）：** 完全离线→本地渲染不留白不转圈；新用户尚未填写宠物名→文案中 `[宠物名]` 占位符替换为「Bayi Berbulu」，档案创建后即时替换为真实名字。
- **宠物身份证：** 为每只宠物档案分配唯一流水 ID 号（全平台自增，是后续稀有号拍卖的基础）；证件卡片样式展示（宠物照片/名字/品种/年龄/ID 号）；分享机制分三级——截图自动带水印保护品牌、链接分享含 App 下载引导、无水印高清图片版本付费下载（Rp5,000）；付费下载复用 GoPay/OVO 支付体系

### 1.2.0（月2 第8-9周）
**兽医端效率升级 + 收费深化**

- **会员制套餐一：** Rp78,000/月（含3次AI问诊 + 2次兽医咨询）

## 月3 · 留存核心功能

### 1.3.0（月3前半）
**健康管理 + 留存视觉钩子 + 裂变拉新**

- **健康提醒激活（会员专属）：** 基于 1.1.0 录入的健康数据，配置提醒间隔——平台预设默认值（狂犬疫苗=每年 / 驱虫=每3个月 / 月经=用户自定义周期），用户可修改；系统自动计算下次提醒时间并触发推送；绝育为一次性事件仅记录不提醒；非会员可录入和查看历史，不触发推送
- **兽医预约功能：** 兽医离线时用户可提交预约请求，兽医上线后推送通知提醒接单；V1.0.0 离线时仅引导回 AI 问诊
- **Tailsonality 宠物人格测试：** TailTopia 原创宠物人格体系；4维度（社交能量/驱动力/情绪表达/生活节奏）× 2极 = 16种角色人格；约15题混合题型（行为观察题+图片选择）；结果角色以 Jaksel 混搭风格命名（如"literally CEO banget"）；分享卡含宠物头像+宠物名+角色名+slogan+一段分析+弱下载引导（复用1.2.0卡片生成框架）；进入方式：档案创建后自动引导 + 成长档案内独立入口；重测：每季度免费一次，额外重测付费（预留D阶段变现）；CP匹配功能预留，等1.6.0 LBS+社交图谱就位后上线；**内容生产需运营在月2启动**（16套插图 + Jaksel文案）
- **首页果冻人气泡（FR-10 · Solo Jelly Bubble）：** 首页视觉核心区（Header 下方、卡片上方）设置单只 3D 拟真果冻宠物气泡；每次冷启动/下拉刷新/从其他 Tab 切回，气泡在安全区内随机位置出现并展示随机宠物头像，制造"每次打开都有小惊喜"的冷启动期待感（FOMO）。
  - **视觉质感（FR-10A）：** 3D 半透明硅胶/玻璃材质，具高光与内部折射；闲置时执行 ±5% 幅度无缝呼吸动效（Lottie/Rive 循环，周期 2000ms）。
  - **随机安全区算法（FR-10B）：** 水平范围 `[r+16px, W-r-16px]`，垂直范围扣除顶部 Header（120px）与底部卡片起始高度（380px）后随机生成；冷启动/下拉刷新/气泡戳破后重进首页均重新随机。
  - **拖拽 & 回弹（FR-10C）：** 任意方向拖拽触发史莱姆拉伸形变，最大形变不超过原始直径 1.5 倍；每拖拽超过 20px 触发一次系统级微震动（iOS `selectionChanged` / Android 微振动）；松手后以弹簧曲线回弹（`damping=12, mass=1.0, stiffness=150`，≤300ms）。
  - **戳破 & 跳转（FR-10D）：** 点击/双击触发粒子溅射动效（150ms）+ 清脆"啵"音效 + 中等级触觉震动；播放完毕后跳转宠物名片（FR-14）或成长档案（FR-37）；宠物已被删除时气泡破裂后弹出 Toast「*Oops, this fur baby went to play elsewhere!*」并重新生成气泡。

- **「一年前的今天」回顾推送：** 用户有历史快乐时刻记录时，在对应日期自动推送回顾通知，附历史内容首图；无记录则不推送

- **用户自定义里程碑：** 用户在平台固定里程碑清单基础上，可追加专属里程碑；平台运营根据用户高频追加内容将其提升为官方里程碑，逐步丰富固定清单

运营侧重点：社区活动运营 · 兽医KOL内容生产 · 付费用户召回 · 会员增长

## 月7+ · 🛍️ 宠物本地服务 O2O

> 独立双边市场业务，工程量显著高于现有功能（约为场地预订的3-4倍）。与 1.7.0 场地预订共享支付和地图基础设施，但属于独立产品线。服务品类优先级：动物医院预约 → 美容护理 → 上门服务 → 训练师 → 宠物摄影 → 寄养/托管（远期）。

### 2.0.0（月7-8）
**O2O MVP · 动物医院预约 + 美容护理到店**

- **线上兽医延伸线下预约（F-B）：** 平台现有线上兽医可开放实体诊所线下就诊预约；用户选择兽医/时间槽/宠物信息提交预约 → 兽医端确认 → 推送通知；复用 1.3.0 兽医预约调度框架；**V2.0.0 线下到诊付款，不含在线支付**
- **美容护理到店预约（A）：** 美容沙龙商家入驻（运营签约保质量 + 商家自助注册保规模）；服务项目：洗澡/修毛/美容/剪指甲；按服务类型和距离筛选；与宠物友好场所共享地图基础设施；用户选择服务/时间槽/宠物信息提交预约 → 商家确认或拒绝 → 推送通知；**线下到店付款**
- **商家基础后台：** 查看预约列表 · 确认/拒绝请求 · 管理服务时间表 · 店铺信息（名称/地址/服务项目/价格/营业时间/照片）

### 2.1.0（月9+）
**实体诊所入驻 + 在线支付 + 电商 Phase B**

- **实体诊所独立入驻（F-A）：** 独立动物医院/诊所账号入驻（区别于线上兽医个人账号）；诊所填写基本信息（地址/科室/坐诊兽医/收费标准/营业时间）；用户可按距离/科室/评分筛选诊所并预约；**需运营在月5-6启动首批诊所签约**
- **在线支付：** 接入 GoPay / OVO（复用 1.1.0 支付体系），平台按交易额抽佣；覆盖美容护理 + 诊所预约定金
- **用户评价系统：** 服务完成后用户评分（1-5星）+ 评价文字；评价公开展示，影响搜索排序
- **商家数据后台升级：** 营收报表 · 评价管理 · 排班管理 · 服务上下架
- **精选电商 Phase B（平台内下单）：** 用户在 App 内完成支付（复用 GoPay/OVO），商家自行履约发货；完整订单系统（下单/支付/物流追踪/售后）；品类扩展至宠物用品；**上线时机由 Phase A 转化数据决定，非固定版本**

### 2.2.0（月11+）
**上门服务 + 公益 Phase 1**

## 数据触发里程碑 · 稀有 ID 拍卖 + NFC 实体宠物证

> **触发条件：DAU ≥ 2万 且 累计新增用户 ≥ 20万**（不绑定固定版本，达到门槛后启动开发，约需 2-3 个月上线）

**定位：** 高消费力宠物主专属高利润虚拟产品，结合实体纪念品，兼顾品牌建设与溢价变现。

**数字拍卖系统**
- **稀有 ID 号规则：** 极低编号（0001-1000）+ 顺子/豹子/回文（如 8888/12345/10001）+ 运营手动标记（联名活动号/周年纪念号）；普通用户身份证为自增流水号，稀有号通过拍卖获得
- **月度专场拍卖：** 定期举办，每场释放限量稀有号；倒计时 + 实时竞价；竞价成功通知 + 落拍提醒；营造事件感和稀缺感
- **拍卖得主权益：** 稀有 ID 号永久绑定该宠物；专属得主徽章展示在主页/身份证；拍卖记录永久存档

**NFC 实体宠物证**
- **卡片内容（NFC 碰触后显示）：** 宠物名片 H5 + 宠物身份证页面 + 拍卖得主专属页（编号证书/竞拍记录/得主自定义内容）
- **自定义内容：** 得主可填写专属寄语/纪念文字，刻入证书页面
- **制作发货：** 平台自洽芯片/印刷合作方，统一制作 NFC 卡片寄送至用户地址（印尼境内）
- **定价：** 拍卖底价 + 实体卡片制作费（定价待商务评估）

**前置准备（达到用户门槛前启动）**
- NFC 芯片 + 卡片印刷合作方评估与签约
- 月度专场运营 SOP 与稀有号释放计划
- 拍卖结算与退款规则法务确认
