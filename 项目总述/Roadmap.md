# PetGo · 6个月产品 Roadmap

**更新日期：** 2026-06-08
**发布节奏：** 每2周一个版本（1.0.0除外）
**支付：** 合作方接入，不占用工程瓶颈
**整体策略：** 月1-2拉新 → 月2开始收费 → 月3-4留存功能全部就位 → 月5运营主导 → 月6收支平衡

## 月1 · 🚀 1.0.0 上线（4-5周）

**目标：** 产品上线，跑通核心两大模块 + 基础社交互动

### 功能清单

- **Auth & Onboarding：** Google登录 · 未登录首页浏览（游客模式，无强制拦截）· 软性登录推荐（浏览至第3页触发）· 昵称确认 · 宠物状态选择（A/B/C）· 宠物档案创建引导（可跳过，前3次重启提示）
- **App 导航：** 5位置底部Tab Bar（首页 / 成长档案 / [+]发布 / 问诊 / 我的）· 未登录点击非首页Tab触发登录引导
- **首页 Feed：** 内容流（时间倒序/无限滚动）· 按宠物状态硬过滤（非算法推荐）· 种子内容冷启动
- **AI 图文分诊：** 照片上传（≤3张/10MB）· 绿/黄/红三级评级 · 条件倒计时协议（黄色）· 半屏强提醒（红色，5秒锁定，不推荐医院）
- **专业问诊：** 兽医聊天（三方SDK）· 兽医账号独立登录入口（账号密码，运营后台创建）· 兽医工作台（待接单/进行中/历史/我的）· 抢单模型（先到先得，后端原子写入）· 1分钟接单超时机制 · 退单机制（最多2次，超出转运营处理）· AI上下文传递给兽医 · 问诊历史记录（AI问诊 + 兽医咨询记录）
- **成长档案：** 宠物档案创建与编辑 · 快乐时刻时间线视图（默认，含第一条记录特殊标记）+ 日历视图（月份格子，有照片显示缩略图）· 里程碑进度条入口 · 档案统计栏（快乐时刻条数/问诊次数）· 问诊记录存档（用户确认）
- **宠物第一次里程碑系统：** 平台固定清单（猫30个 / 狗30个 / 其他宠物15个）· 三级庆祝动效（S半屏 / M全屏+徽章 / L开宝箱式）· 徽章图标打卡交互（关联已发布内容或去发布）
- **宠物名片 H5：** 对外分享页 · Hero区（陪伴天数）· 里程碑徽章条 · 故事数字 · 最近里程碑动态 · 最近5条快乐时刻照片流 · 双CTA（有宠/无宠差异化）· Open Graph标签
- **内容发布：** 统一「+」入口 · 三类内容（日常分享/专业科普/成长日历）· 图片上传（每帖≤9张/单张≤10MB）· 1000字符上限 · 三方自动审核（文字关键词+图片识别，秒级拦截）
- **内容详情页：** 全屏详情 · 迷你主页预览卡（底部弹出，不跳转新页）
- **内容互动：** 点赞（可取消）· 一级评论 + 回复评论（两级）· 内容删除（作者可删，V1.0.0 无编辑功能）· 内容举报（5类选项，进运营审核队列）
- **个人中心（我的）：** 用户信息（头像/昵称可修改）· 宠物卡片 · 我的发布（三类内容时间线倒序）· 宠物状态修改 · 账号注销（印尼PDP合规，二次确认，不可恢复）· 退出登录
- **推送通知：** 兽医回复提醒 · 兽医端新请求提醒 · 被点赞/评论提醒 · 宠物生日提醒（生日前1天）· 陪伴纪念日提醒（满30/100/365天）· L级里程碑解锁提醒 · 权限申请时机：首次问诊后 OR 档案创建后取最早（仅申请一次）
- **系统功能：** App内通知中心（铃铛）· 语言切换（印尼语/英语）· 深链接（7种推送类型全覆盖直跳目标页）· App版本更新提醒（App内弹窗，无需推送权限）
- **兽医评分系统：** 问诊结束后用户评分（1-5星）· 30分钟保护窗口 · 未评分进入问诊Tab时弹出补评一次

> **WhatsApp / SMS OTP登录：** 根据上线后注册转化率数据决定上线时机，不固定在某版本

### 核心指标
- 下载量 · 注册用户数 · AI问诊次数 · 宠物名片分享→下载转化率 · D7留存率

## 月2 · 收费启动

### 1.1.0（月2 第6-7周）
**支付 + 收费正式上线**

- **支付接入：** GoPay / OVO（合作方接入）
- **AI问诊收费：** Rp18,000/次；**每个月第一次AI问诊免费**（降低首次尝试门槛）
- **兽医咨询收费：** Rp30,000/次；AI问诊费可抵扣，升级补差价 Rp12,000
- **Apple登录：** iOS App Store要求
- **健康档案数据录入（早期版）：** 用户记录健康事件类型（疫苗/驱虫/月经/绝育/自定义）+ 上次时间；数据存储供 1.3.0 提醒功能使用；录入成功后提示「开通会员后自动提醒下次时间」，不留歧义
- **成长档案月度回顾：** 每月自动生成月度回顾卡片，汇总当月快乐时刻数、里程碑完成数、最受欢迎照片；用户进入成长档案时展示「[宠物名] 的 X 月」；可一键分享
- **新用户快速解锁里程碑（6个）：** 复用 1.0.0 里程碑庆祝框架（三级动效+徽章），新增6个触发条件，工程量极小（约2-3天）；目标是让用户在前两周内积累成就感，提升付费转化率
  - 发布第一篇内容（Day 1-3）
  - 完成第一次AI问诊（Day 1-7）
  - 宠物档案完整度100%（Week 1）
  - 记录第5条快乐时刻（Week 1-2）
  - 获得第一个点赞（Day 1-7）
  - 在平台待满7天（注册后第7天自动触发）

### 1.2.0（月2 第8-9周）
**兽医端效率升级 + 收费深化**

- **AI辅助起草回复（兽医端）：** AI为兽医生成参考回复，兽医确认后发送；差异化分成调整为 50/50
- **多宠物账号管理：** 单账号管理多只宠物，成长日历必须选择归属宠物（试用，后续纳入额外宠物收费）
- **宠物年龄换算分享卡：** 将宠物年龄换算为人类年龄当量，生成精美卡片一键分享至 WhatsApp / Instagram Stories；猫用科学公式（前2年各≈15人岁，之后每年≈4人岁），狗按体型分档（小/中/大/超大型，生成时用户选择），其他宠物用通用近似并注明仅供参考；目标驱动裂变拉新
- **会员制套餐一：** Rp78,000/月（含3次AI问诊 + 2次兽医咨询）

### 核心指标
- 付费转化率 · 首次→付费比例 · 兽医平均评分 · 会员订阅数

## 月3 · 留存核心功能

### 1.3.0（月3前半）
**健康管理 + 多宠物（会员权益绑定）**

- **健康提醒激活（会员专属）：** 基于 1.1.0 录入的健康数据，配置提醒间隔——平台预设默认值（狂犬疫苗=每年 / 驱虫=每3个月 / 月经=用户自定义周期），用户可修改；系统自动计算下次提醒时间并触发推送；绝育为一次性事件仅记录不提醒；非会员可录入和查看历史，不触发推送
- **兽医预约功能：** 兽医离线时用户可提交预约请求，兽医上线后推送通知提醒接单；V1.0.0 离线时仅引导回 AI 问诊

### 1.4.0（月3后半）
**内容生态完善 + 宠物友好场所发现 MVP**

- 收藏功能
- 内容编辑（已发布的内容进行文字修改）
- 评论举报
- **内容社区视频发布：** 用户在 Feed 中发布视频内容；依赖完整视频转码 + CDN 分发基础设施
- **推送通知分类开关：** 点赞/评论/兽医回复/宠物健康推送提醒可单独开关；V1.0.0 为全开或全关
- **内容分享卡片：** 内容详情页一键生成图片卡片（含内容首图/文字摘要/作者/App品牌水印），通过系统分享菜单发至 WhatsApp / Instagram Stories 等；非 App 用户看到静态图片 + 下载引导；复用年龄换算卡片的生成框架
- **宠物友好场所发现 MVP：** 用户标记场所（名称/类型/照片/宠物友好标签）· 列表+地图浏览附近场所 · 场所详情卡（基本信息+用户照片+评论）· 运营预置冷启动数据（雅加达核心场所）
- **会员制套餐二：** Rp108,000/月（含3次AI问诊 + 1次兽医咨询 + 三只及以上宠物 + 定期推送提醒）

### 核心指标
- 健康提醒设置率 · 会员留存率 · D30留存率 · 内容发布频率

## 月5 · 🏠 运营主导留存

产品侧进入支持角色：
- Bug修复与性能优化
- A/B测试配合运营验证留存假设
- 数据看板完善
- 小功能迭代（根据用户反馈）
- **摄影师署名 + 同框记录：** 攒局活动中专业摄影师可标记作品，宠物主可认领同框照片并存入档案
- Feed算法优化

**成长档案留存深化（产品配合运营）：**

- **「一年前的今天」回顾推送：** 用户有历史快乐时刻记录时，在对应日期自动推送回顾通知，附历史内容首图；无记录则不推送
- **内容连续记录感知：** 档案页展示「本月已记录 X 次」，低压力频率感知，不做 streak 打卡机制
- **生日纪念卡自动生成：** 宠物生日当天自动生成可一键分享的纪念卡（含宠物头像 + 年龄 + 精选照片），用户分享至 WhatsApp / Instagram
- **里程碑分享卡：** 陪伴满100天、365天等节点，自动生成可分享卡片，作为用户自发传播的新素材
- **用户自定义里程碑：** 用户在平台固定里程碑清单基础上，可追加专属里程碑；平台运营根据用户高频追加内容将其提升为官方里程碑，逐步丰富固定清单

运营侧重点：社区活动运营 · 兽医KOL内容生产 · 付费用户召回 · 会员增长

## 月6 · 💰 收支平衡

### 1.7.0（月6前半）
- AI健康趋势预警（成长档案数据积累后启用）
- 品牌合作基础功能（赞助商曝光位）
- **三方广告网络接入（MAU ≥ 1万后触发）：** 接入 Meta Audience Network 或 Google AdMob，在非会员用户的信息流中插入原生广告；付费会员全程无广告；广告格式以原生信息流为主，与社区内容视觉风格保持一致
- **平台场地预订与托管支付：** 用户通过 App 预订攒局场地并完成在线支付，平台按交易额抽佣；需接入场地方合作网络
- **宠物友好场所：接入三方数据（Google Maps / Foursquare）**：与场地预订合并为「地点生态」模块；场所数据量级大幅提升，不再依赖纯 UGC

### 1.8.0（月6后半）
- **视频AI问诊：** 收费升级版，付费/会员专属
- 宠物档案导出/备份
- **个人兽医注册认证体系：** 兽医可个人注册入驻（非机构派驻），含资质认证、阶梯分成（青铜/黄金/钻石）和个人主页；替代当前机构合作模式

## 月7+ · 🛍️ 宠物本地服务 O2O

> 独立双边市场业务，工程量显著高于现有功能（约为场地预订的3-4倍）。与 1.7.0 场地预订共享支付和地图基础设施，但属于独立产品线。运营需在**月5**启动首批商家签约，否则 2.0.0 上线时无商家可展示。

### 2.0.0（月7-8）
**宠物护理预约 MVP**

- **商家入驻：** 运营签约（保质量）+ 商家自助注册（保规模）两种方式；商家填写店铺信息（名称/地址/服务项目/价格/营业时间/照片）
- **服务浏览：** 按服务类型（洗澡/修毛/美容/宠物剪指甲等）和距离筛选；与宠物友好场所共享地图基础设施
- **在线预约：** 用户选择服务/时间槽/宠物信息提交预约 → 商家端确认或拒绝 → 用户收到推送通知；**V2.0.0 暂不含在线支付，线下到店付款**
- **商家基础后台：** 查看预约列表 · 确认/拒绝请求 · 管理服务时间表

### 2.1.0（月9+）
**O2O 完整版**

- **在线支付：** 接入 GoPay / OVO（复用 1.1.0 支付体系），平台按交易额抽佣
- **用户评价系统：** 服务完成后用户评分（1-5星）+ 评价文字；评价公开展示，影响商家搜索排序
- **商家数据后台升级：** 营收报表 · 评价管理 · 排班管理 · 服务上下架
