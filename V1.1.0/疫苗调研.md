# 宠物健康提醒事件调研 — 印尼本地化版本

**调研日期**：2026-06-29
**适用市场**：印度尼西亚（Indonesia）
**适用物种**：犬（Anjing）/ 猫（Kucing）

---

## 一、印尼法规背景

### 狂犬病现状（2024）
- 印尼全国 **26个省份** 属狂犬病流行地区（endemic），仅少数省为无狂犬病区
- 2023年东努沙登加拉省（NTT）记录约 **19,320** 例疑似狂犬病动物咬伤人类案例
- 2024年1-7月已有 **71人** 死于狂犬病；巴厘岛2024年仅一年咬伤案例超 **39,000** 件
- 印尼政府目标：**2030年消灭狂犬病**，正推进爪哇、巴厘、西努沙登加拉、东努沙登加拉的大规模犬类疫苗接种计划

### 无狂犬病区（禁止携带宠物进入）
以下地区目前属于无狂犬病清洁区，**禁止将宠物从流行区带入**：
- 马都拉岛（Madura）
- 巴厘岛（Bali）*（现已感染，仍属特别管控区）*
- 西努沙登加拉（Nusa Tenggara Barat）
- 加里曼丹西部（Kalimantan Barat）
- 马鲁古（Maluku）
- 巴布亚（Irian Jaya）
- 苏门答腊周边岛屿

### 核心法规要求
- 狂犬病疫苗：**法定强制接种**（宠物犬猫、猴子均须接种，3月龄起）
- 每年须接种加强针，不得中断
- 进省/跨岛携宠需携带疫苗接种证明

## 二、疫苗提醒事件

### 犬类（Anjing）

| 疫苗名称 | 本地名称 | 类别 | 首次接种时间 | 后续频率 | 提醒提前天数 | 印尼特别说明 |
|----------|----------|------|-------------|----------|------------|------------|
| 狂犬病疫苗 | Vaksin Rabies | **法规强制** | 3月龄首针 | 每年1次 | 提前14天 | 26省强制；跨省需携带证明 |
| 犬六联/七联（DHPP+L） | Vaksin Combo | 核心推荐 | 6-8周/10-12周/14-16周，共3针 | 每年1次加强 | 提前7天 | 钩端螺旋体菌（Leptospirosis）项在印尼尤为重要 |
| 犬窝咳（Bordetella） | Vaksin Kennel Cough | 非核心 | 宠物店/寄宿犬建议接种 | 每6-12个月 | 提前7天 | 印尼城市宠物店密度高，接触风险较大 |
| 犬流感 | Vaksin Influenza Anjing | 非核心 | 按需 | 每年1次 | 提前7天 | — |

**幼犬首次接种系列时间线：**
```
6-8周  → 第1针（犬六联）
10-12周 → 第2针（犬六联）
14-16周 → 第3针（犬六联 + 狂犬病）
→ 进入年度维护阶段
```

### 猫类（Kucing）

| 疫苗名称 | 本地名称 | 类别 | 首次接种时间 | 后续频率 | 提醒提前天数 | 印尼特别说明 |
|----------|----------|------|-------------|----------|------------|------------|


## 三、驱虫提醒事件

### 印尼热带气候的特殊背景
- 印尼寄生虫感染率极高：研究显示茂物（Bogor）地区 **61%的犬** 和 **53.8%的猫** 携带蠕虫（钩虫、弓蛔虫、绦虫等）
- 蚊子密度高（Aedes aegypti、Culex等），**心丝虫** 感染率显著高于温带国家（东南亚部分地区感染率达16-24%）
- 雨季（10-4月）寄生虫活跃度大幅上升

### 体内驱虫（Obat Cacing）

| 目标寄生虫 | 幼年期（<6月龄） | 成年期（印尼建议） | 产品类型 | 提醒频率 |
|-----------|----------------|-----------------|---------|---------|
| 蛔虫/钩虫/绦虫 | 每2周1次 → 每月1次 | 每**2-3个月**1次（热带气候建议缩短至2个月） | 口服片/糊剂 | 固定周期 |
| 心丝虫预防（Cacing Jantung） | 2月龄起每月1次 | **每月1次**（印尼强烈建议） | 口服/滴剂 | 每月固定日 |

> **重要**：心丝虫一旦感染治疗成本极高且风险大，在印尼应作为**核心必提醒项**。

### 体外驱虫（Obat Kutu/Parasit Luar）

| 驱虫对象 | 印尼威胁等级 | 产品类型 | 建议频率 | 提醒说明 |
|---------|-----------|---------|---------|---------|
| 跳蚤（Kutu Anjing/Kucing） | 高 | 滴剂/喷剂 | 每月1次 | 月度固定提醒 |
| 蜱虫（Caplak） | **极高** | 滴剂/项圈（8个月有效） | 每月/项圈到期提醒 | 传播艾利希体病（Ehrlichia）、巴贝西虫病（Babesia），印尼高发 |
| 耳螨（Tungau Telinga） | 中 | 耳药水 | 每月检查，每3月预防性处理 | — |
| 皮肤螨（Mange） | 中 | 药浴/注射 | 按感染情况 | 印尼湿热气候螨虫繁殖快 |

### 印尼蜱虫传播的特有疾病（产品定位参考）
| 疾病 | 病原体 | 症状 | 预防方式 |
|------|--------|------|---------|
| 犬艾利希体病（Ehrlichiosis） | Ehrlichia canis | 发热、出血、贫血 | 蜱虫驱虫 |
| 犬巴贝西虫病（Babesiosis） | Babesia gibsoni | 溶血性贫血 | 蜱虫驱虫 |
| 钩端螺旋体病（Leptospirosis） | Leptospira spp. | 肾衰、黄疸（可传人） | 疫苗+环境控制 |

## 四、常规健康护理（非疫苗/驱虫）

| 护理事项 | 建议频率（印尼气候调整版） | 备注 |
|---------|----------------------|------|
| 年度全身体检 | 每年1次；7岁以上每6个月 | 包含血常规、尿检 |
| 牙齿检查 | 每年1次（洗牙） | 家庭刷牙每周 |
| 洗澡/美容 | 每2-4周（印尼湿热气候比温带频率高） | 可自定义 |
| 指甲修剪 | 每3-4周 | |
| 耳朵清洁 | 每1-2周（垂耳犬更频繁；雨季后增加频率） | 潮湿气候易感染耳道 |
| 眼睛护理 | 每周 | 短鼻犬种（斗牛/京巴）更重要 |

## 五、一次性里程碑事件

| 事项 | 建议时间节点 | 触发方式 |
|------|------------|---------|
| 绝育/结扎（Sterilisasi） | 犬猫5-7月龄 | 基于出生日期自动推算提醒 |
| 芯片植入（Microchip） | 3月龄以上 | 登记宠物信息时推送提示 |
| 幼年疫苗系列完成通知 | 第3针接种后 | 系统提示"进入年度维护阶段" |
| 老年期转变提醒 | 犬7岁/猫10岁 | 推送换粮、体检频率调整建议 |

## 六、季节性提醒（印尼特有）

| 季节 | 时间 | 推送内容 |
|------|------|---------|
| 雨季前 | 10月、3月（两次雨季前） | 加强体外驱虫（蜱/跳蚤）提醒；检查耳朵/皮肤 |
| 旱季 | 5-9月 | 注意中暑/散热；减少户外活动提示 |
| 全年每月 | 固定日期 | 心丝虫预防药提醒 |

## 七、产品设计建议

### 提醒优先级分层
| 优先级 | 事项 | 原因 |
|--------|------|------|
| P0（必提醒） | 狂犬病疫苗年度加强 | 印尼法规强制；26省流行 |
| P0（必提醒） | 心丝虫月度预防药 | 印尼热带高发，治疗代价极高 |
| P1（核心提醒） | 犬六联/猫三联年度加强 | 核心保护 |
| P1（核心提醒） | 体内驱虫（每2-3月） | 印尼感染率超60% |
| P1（核心提醒） | 体外驱虫（每月） | 蜱虫传播疾病高风险 |
| P2（可选提醒） | 洗澡/美容/指甲/耳朵 | 用户自定义频率 |
| P2（可选提醒） | 年度体检 | 提前30天提醒 |

### 健康提醒数据结构（参考）
```
HealthEvent {
  id: string
  type: "vaccine" | "deworming_internal" | "deworming_external" | "checkup" | "grooming" | "milestone"
  petSpecies: "dog" | "cat"
  name_id: string          // 印尼语名称
  name_en: string          // 英语名称
  firstDate: date          // 首次提醒日期（基于出生日计算）
  recurrence: {
    unit: "week" | "month" | "year"
    interval: number
  } | null                 // null = 一次性事件
  reminderLeadDays: number // 提前N天推送提醒
  isLegallyRequired: boolean
  isTropicalRiskFactor: boolean  // 印尼热带气候特有风险项
  seasonalBoost: string[] | null // 特定季节加强提醒
}
```
